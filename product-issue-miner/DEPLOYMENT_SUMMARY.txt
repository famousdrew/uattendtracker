QUICK START
================================================================================

1. Read RAILWAY_QUICKSTART.md (5 minutes to understand)

2. Create Railway Project
   - Go to https://railway.app
   - Click "Create New Project"
   - Select "Deploy from GitHub"

3. Add Services in Order
   Step 1: Add PostgreSQL (manages DATABASE_URL)
   Step 2: Add Redis (manages REDIS_URL)
   Step 3: Deploy Backend with environment variables
   Step 4: Deploy Frontend with API URL

4. Set Environment Variables
   Backend: ZENDESK_API_TOKEN, ANTHROPIC_API_KEY, DASHBOARD_PASSWORD
   Frontend: NEXT_PUBLIC_API_URL

5. Verify Deployment
   Test: https://<backend-url>/health
   Visit: https://<frontend-url>

Full details: See RAILWAY_QUICKSTART.md and DEPLOYMENT.md
